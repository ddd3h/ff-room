# 数値モデルの仮定と境界条件

研究用途のための再現性確保を目的として、本ツールの数値モデルが置いている
仮定・近似・数値手法をすべてここに記載する。

---

## 1. 支配方程式

### 非圧縮性 Navier-Stokes 方程式 (Boussinesq近似)

```
運動量方程式:  ∂u/∂t + (u·∇)u = -(1/ρ)∇p + ν∇²u + f_buoy
質量保存:     ∇·u = 0
温度方程式:   ∂T/∂t + (u·∇)T = α∇²T
浮力体積力:   f_buoy = (0, 0, g·β·(T_face - T_ref))   [+z = 鉛直上方]
```

| 記号 | 意味 | 単位 |
|------|------|------|
| u = (u, v, w) | 速度ベクトル | m/s |
| p | 圧力 (修正圧力) | Pa |
| ρ | 密度 | kg/m³ |
| ν | 動粘度係数 | m²/s |
| T | 温度 | °C (内部計算は °C のまま, T_ref は K 換算) |
| α | 温度拡散係数 = k/(ρ·Cp) | m²/s |
| g | 重力加速度 = 9.81 | m/s² |
| β | 体積膨張係数 = 1/(T_initial + 273.15) | 1/K |
| T_ref | 参照温度 = T_initial | °C |

`buoyancy: false` (デフォルト) のとき f_buoy = 0 (等温モデル)。

## 2. 物理モデルの仮定

### 仮定 1: 非圧縮性流れ
密度 ρ = 定数 = 1.2 kg/m³ (20°C, 1 atm の空気)。  
Ma << 1 (マッハ数 ≪ 1) が成立するとき有効。  
室内気流の典型風速 1〜5 m/s に対し Ma ≈ 0.004〜0.015 → 仮定は妥当。

### 仮定 2: 弱圧縮性 (Boussinesq近似)
密度変化は浮力項のみに影響させ、その他の項では ρ = 定数 とする。  
`buoyancy: false` のときは完全等温 (温度方程式も不活性)。  
`buoyancy: true` のときは温度移流拡散 + 鉛直浮力が有効になる。

### 仮定 3: ニュートン流体
応力 = 動粘度 × 変形速度テンソル。  
ν = 1.5×10⁻⁵ m²/s (20°C, 空気の動粘度)。

### 仮定 4: 乱流モデルなし
直接数値シミュレーション (DNS) 的なアプローチ。  
格子スケール以下の乱流構造は解像しない。

セル Reynolds 数の推定:

```
Re_cell = U_max × dx / ν
```

`U_max=3 m/s, dx=0.2m, ν=1.5e-5` → Re_cell ≈ 40,000 → 粗い格子では乱流は「数値的に消散」している。  
結果は定性的な流れパターンとして解釈する。

精密な乱流解析が必要な場合: Smagorinsky LES モデルの追加を developer_guide.md 参照。

### 仮定 5: 剛体壁
壁・障害物はすべて no-slip (速度ゼロ) の剛体面として扱う。
壁面の粗さ・熱伝達は考慮しない。

## 3. 数値手法

### 3-1. 離散化: MAC スタガード格子

**Harlow & Welch (1965)** の Marker-And-Cell (MAC) 法に基づく。

速度とスカラーを格子の異なる位置に置くことで、
圧力振動 (チェッカーボード不安定) を回避する。

```
u[i,j,k] : x方向速度 — x面中心 (i = 0..Nx,   j = 0..Ny-1, k = 0..Nz-1)
v[i,j,k] : y方向速度 — y面中心 (i = 0..Nx-1, j = 0..Ny,   k = 0..Nz-1)
w[i,j,k] : z方向速度 — z面中心 (i = 0..Nx-1, j = 0..Ny-1, k = 0..Nz  )
p[i,j,k] : 圧力      — セル中心 (i = 0..Nx-1, j = 0..Ny-1, k = 0..Nz-1)
```

セルサイズ: dx = Lx/Nx, dy = Ly/Ny, dz = Lz/Nz (均一格子)。

### 3-2. 時間積分: 前進 Euler (1次精度)

```
u^{n+1} = u^n + dt * RHS(u^n)
```

陽的スキームのため CFL 条件を満たす dt が必要。

### 3-3. 移流項: 1次精度 Upwind

```
u ∂φ/∂x ≈ u × (φ_i - φ_{i-1})/dx  (u > 0)
           u × (φ_{i+1} - φ_i)/dx  (u < 0)
```

**特性**: 無条件単調、数値拡散が強い (有効拡散 ≈ |u|×dx/2)。  
高精度化: QUICK / Lax-Wendroff への切り替えは developer_guide.md 参照。

### 3-4. 拡散項: 2次精度中心差分

```
ν ∂²u/∂x² ≈ ν × (u_{i+1} - 2u_i + u_{i-1}) / dx²
```

### 3-5. 射影法 (Chorin 1968)

各タイムステップ:

```
ステップ 1 (中間速度):
    u* = u^n + dt × [-(u^n·∇)u^n + ν∇²u^n]

ステップ 2 (圧力 Poisson 方程式):
    ∇²p^{n+1} = (ρ/dt) ∇·u*

ステップ 3 (速度補正):
    u^{n+1} = u* - (dt/ρ) ∇p^{n+1}
    ∇·u^{n+1} ≈ 0 を保証
```

### 3-6. Poisson ソルバー: 共役勾配 (CG) 法

- 前処理なし (将来: Jacobi 前処理または不完全 LU)
- 初期推定: p = 0
- 収束判定: ||r||₂ / ||b||₂ < tol (デフォルト tol = 1×10⁻⁶)

## 4. 境界条件

### 部屋の壁 (6面)

| 壁面 | 速度条件 | 圧力条件 |
|------|---------|---------|
| 西壁 (x=0) | u=0, v=0, w=0 (no-slip) | ∂p/∂n=0 (Neumann) |
| 東壁 (x=Lx) | u=0, v=0, w=0 (no-slip) | ∂p/∂n=0 (Neumann) |
| 南壁 (y=0) | u=0, v=0, w=0 (no-slip) | ∂p/∂n=0 (Neumann) |
| 北壁 (y=Ly) | u=0, v=0, w=0 (no-slip) | ∂p/∂n=0 (Neumann) |
| 床 (z=0) | u=0, v=0, w=0 (no-slip) | ∂p/∂n=0 (Neumann) |
| 天井 (z=Lz) | u=0, v=0, w=0 (no-slip) | ∂p/∂n=0 (Neumann) |

### 障害物 (SOLID セル)

SOLIDセルの隣接する全速度面にゼロを設定 (no-slip)。  
圧力: Neumann (∂p/∂n=0)。  
障害物内部の圧力は求解しない (方程式から除外)。

### 扇風機 (INFLOW セル)

INFLOW セルの face に設定風速 `U_fan` を Dirichlet 条件として毎ステップ上書き。  
圧力: FLUID と同じ Poisson を解く (圧力は拘束しない)。

### 流出境界 (OUTFLOW)

デフォルト: なし (全壁が no-slip)。  
OUTFLOWセルを追加した場合:
- 速度: ゼロ勾配 (∂u/∂n = 0, 外挿)
- 圧力: Dirichlet (p = 0)

### 開口部 (Opening: 窓・ドア)

開口部は OutflowBC (p=0 Dirichlet) + 温度交換の複合境界条件。

```
速度条件: OUTFLOW と同じ (p=0)
温度条件:
  法線方向速度 u_f の符号を確認
  流入 (u_f が外→内) → T = T_outside   (外気温を Dirichlet で設定)
  流出 (u_f が内→外) → ∂T/∂n = 0      (室内温度のゼロ勾配)
  u_f = 0 (起動時) → T = T_outside     (安全側: 外気温を適用)
```

各開口部に個別の `T_outside` を設定可能 (例: 南窓と北窓で外気温が異なる場合)。  
`T_outside` 省略時は `solver.T_outside` を使用。

**自動アウトフローの無効化**: 開口部が1つ以上設定されている場合、
scene.py は自動アウトフロー (内壁全面を OUTFLOW とするデフォルト動作) を無効にする。
自動アウトフローは外気温を持たないため、開口部と共存すると冷却効果が相殺される。

## 5. 扇風機モデルの近似

| 実物 | モデル |
|------|--------|
| 円形断面 | 正方形パッチ (辺 ≈ 2×radius) |
| 非均一速度プロファイル | 一様流入 (全面同一速度) |
| 回転に伴うスワール | 無視 (直線吹き出しのみ) |
| 斜め向き (MVP) | 最近接軸にスナップ |
| 広がり角 | 無視 (後で拡張予定) |

## 7. 安定性条件 (CFL)

時間ステップ `dt` は以下を満たす必要がある:

```
移流安定性: dt < CFL × min(dx, dy, dz) / U_max    (CFL ≤ 1 推奨)
拡散安定性: dt < 0.5 × min(dx², dy², dz²) / ν
```

**例**: 5×4×2.5m 部屋, 25×20×12 格子 (dx=0.2m), U_max=3 m/s

| 条件 | 上限 dt |
|------|---------|
| 移流 (CFL=0.5) | 0.033 s |
| 拡散 | 1333 s |
| **推奨** | **0.01 s** |

## 8. 既知の制限事項

1. **乱流モデルなし**: 格子スケール以下の渦は解像されない。結果は定性的指標として使用する
2. **1次 Upwind 移流**: 数値拡散により、鋭い流れ境界が拡散する (温度場も同様)
3. **扇風機の一様流入近似**: 実際の速度プロファイルと異なる
4. **陽的時間積分**: 大きい dt で発散する (CFL 条件)
5. **Boussinesq近似**: 密度変化を浮力のみに限定。大きな温度差 (>20°C) では誤差が増える
6. **壁面の断熱近似**: 壁・障害物・天井の熱伝導は考慮しない。実際より冷却が早い
7. **軸平行な方向のみ** (MVP): 扇風機の斜め方向は未対応
8. **均一格子**: 壁面境界層を細かく解像したい場合は細かい格子全体が必要

## 9. 再現性の保証

設定ファイル (YAML/JSON) から完全に再現できる:

```bash
# 同じ設定ファイルから常に同じ結果が得られる
python examples/run_basic.py
```

- 乱数使用なし → 決定論的
- 設定ハッシュを実験ログに記録 → トレーサビリティ
- 設定ファイルに `metadata.date`, `metadata.author` フィールドあり
